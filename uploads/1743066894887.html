📧 Temp Mail API

Easily generate temporary emails & fetch inbox messages.

📌 Overview

The Temp Mail API lets you generate temporary email addresses and fetch received messages. This documentation provides details on all available API endpoints.

🌍 Base URL

https://tempmail.bjcoderx.workers.dev/

🔹 API Endpoints

1️⃣ Generate Temporary Email

📌 Endpoint: /gen

📜 Method: GET

📄 Description: Generates a new random temporary email.

📤 Response:

{
    "status": "ok",
    "mail": "xyz123@osxofulk.com",
    "expired_at": 1738992113,
    "join": "@BJ_Devs on Telegram"
}

2️⃣ Generate Custom Email

📌 Endpoint: /gen?name=customname

📜 Method: GET

📄 Description: Generates a temp email with a custom username.

📤 Response:

{
    "status": "ok",
    "mail": "customname@osxofulk.com",
    "expired_at": 1738992113,
    "join": "@BJ_Devs on Telegram"
}

3️⃣ Fetch Inbox

📌 Endpoint: /inbox/{email}

📜 Method: GET

📄 Description: Retrieves emails received by the given temp email.

📤 Response:

{
    "status": "ok",
    "mail": "xyywiu2myn@daouse.com",
    "messages": [
        {
            "subject": "514527 is your code to confirm this email address",
            "from": "Facebook security@facebookmail.com",
            "join": "@BJ_Devs on Telegram"
        }
    ]
}
